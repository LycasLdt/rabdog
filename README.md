# :dog:

## Rabdog 
</div>

快速获取 **Scratch社区作品**。

> [!WARNING]
> 仍处于**超前版本**，请谨慎使用
>
> 凡使用此程序**违反社区规定**的，此程序**不承担任何责任**

### :rocket:支持
* :white_check_mark: [**CCW**][ccw]
  全部支持
* :white_check_mark: [**Clipcc**][clipcc]
  全部支持
* :white_check_mark: [**小码王**][xmw]
  全部支持
* :white_check_mark: [**Scratch 中社**][scratch-cn]
  从`v0.1.2`开始支持

### :white_check_mark:使用

```bash
rabdog [SOURCE] <OPTIONS>
```
__SOURCE__: **社区作品**链接

__OPTIONS__:

`可选` `-p, --path <PATH>`: `.sb3`文件存储路径
